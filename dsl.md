## Patterns

- pulse(hits, steps = 16) 


## Mods


## Adding tracks

```
# = track id

#/[voice] [...pattern] [...mods]

Examples:

0/kick pulse(4,16) 
1/hat pulse(16,16)
```

## Commands to tracks

```
# = track id

#/start - start if stopped
#/stop - stop if started
#/mod [...mods] - add 1 or more mods

Examples: 

0/start
1/stop


```
